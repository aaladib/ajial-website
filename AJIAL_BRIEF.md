## Projects Section
Create a professional projects/gallery section.

Important:
- Do not invent fake project names.
- Do not invent fake client names.
- Do not invent fake numbers.
- Use placeholder project cards that can be replaced later.
- Each project card should support:
  - Project image
  - Project title
  - Project sector
  - Short description
  - Location
  - Status

Use neutral placeholders such as:
- مشروع أعمال معدنية
- مشروع مقاولات إنشائية
- مشروع تشطيبات
